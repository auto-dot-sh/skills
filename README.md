# auto skills

Agent skills for [auto](https://auto.sh) — program software factories the
same way you program CI/CD.

```sh
npx skills add auto-dot-sh/skills
```

## Skills

| Skill | Description |
| --- | --- |
| [auto-onboarding](skills/auto-onboarding/SKILL.md) | Takes a repository from zero to a working auto deployment in one conversation: pitch, channel, dossier, first contact, one workflow, CI apply on merge. |

Agents without skill support get the same playbook from the CLI:
`auto onboard --agent`.

## Contributing

This repository is a read-only mirror synced by CI from the auto monorepo;
each SKILL.md carries the source commit in its frontmatter. Issues and pull
requests opened here are not monitored.
